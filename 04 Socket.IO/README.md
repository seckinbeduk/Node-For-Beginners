#Repository for Beginners

This is the repository, containing the example source code of Node.JS

## Socket.IO Chat

A simple chat demo for socket.io

#### How to use

```
$ cd 04 Socket.IO
$ npm install
$ npm start
```
And point your browser to `http://localhost:3000`.
